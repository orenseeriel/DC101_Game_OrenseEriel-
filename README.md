## DC101_Game_Orense-Eriel

## Snake Game 
This is a simple web game Snake Game built using HTML, CSS, and JavaScript.


## How to Run the Game Using Visual Studio

Requirements
Visual Studio Code
A modern web browser (Chrome, Edge, Firefox)

## Steps
Download the Zip file

Find the DC101_Game_OrenseEriel-main.zip on the Downloads

Right click and select Extract Here

Once the extracting is done, open Visual Studio Code

Click File → Open Folder

Select the project folder: DC101_Game_WatiwatAlexander-main

Locate the file index.html

Right-click on index.html and choose:

Open with Live Server (recommended), or
Open in Default Browser

## Gameplay Instructions

Press any key to start Use W,A,S,D or Arrow Keys to move and if 
Game Over! Press Space to Restart
